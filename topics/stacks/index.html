What Is a Stack?

The One-Sentence Answer

A stack is a Last-In, First-Out (LIFO) data structure — the last item you put in is the first one that comes out.

Real-World Analogy

A stack of plates. You add plates on top. You take plates from the top. You can't pull from the middle or bottom without everything crashing.

Also: Ctrl+Z (Undo) — your actions stack up, and undo peels off the most recent one first. Browser back button — pages stack up, back pops the latest.

Under the Hood — Implementation

Array-Based Stack (Most Common)

class Stack {
  constructor() {
    this.items = [];
  }
  push(val) { this.items.push(val); }
  pop()     { return this.items.pop(); }
  peek()    { return this.items[this.items.length - 1]; }
  isEmpty() { return this.items.length === 0; }
  size()    { return this.items.length; }
}
In JavaScript interviews: Just use a plain array with .push() and .pop(). No need for a class. Arrays ARE stacks in JS.

Linked-List-Based Stack

class LLStack {
  constructor() { this.head = null; this.count = 0; }

  push(val) {
    const node = { val, next: this.head };
    this.head = node;
    this.count++;
  }
  pop() {
    if (!this.head) return undefined;
    const val = this.head.val;
    this.head = this.head.next;
    this.count--;
    return val;
  }
  peek() { return this.head?.val; }
}
When to use linked-list stack? When you need guaranteed O(1) (no amortized resizing). In practice, array-based is almost always better due to cache locality.

Pattern: Valid Parentheses

The Classic Interview Problem

Given a string with ( ) [ ] { }, determine if it's valid — every open bracket has a matching close bracket in the correct order.

Real-World

Nesting Russian dolls. The last doll you opened must be the first one you close. You can't close the big doll before closing the small one inside it.

Code: Valid Parentheses

function isValid(s) {
  const stack = [];
  const map = { ')': '(', ']': '[', '}': '{' };

  for (const char of s) {
    if (char === '(' || char === '[' || char === '{') {
      stack.push(char);            // opening → push
    } else {
      if (stack.pop() !== map[char]) // closing → must match top
        return false;
    }
  }

  return stack.length === 0; // must be empty at end
}
Why stack? The LAST opening bracket must match the FIRST closing bracket — that's exactly LIFO. If you see ( then [, the [ must close before (.

Pattern: Monotonic Stack

What Is It?

A stack that maintains increasing or decreasing order. Before pushing, pop anything that violates the order. This lets you efficiently find the "next greater" or "next smaller" element.

Real-World

Standing in a line looking forward. A short person can see the next taller person ahead. Everyone between them (shorter people) is irrelevant. A monotonic stack removes those irrelevant elements.

Code: Next Greater Element

function nextGreaterElement(nums) {
  const result = new Array(nums.length).fill(-1);
  const stack = []; // stores INDICES

  for (let i = 0; i < nums.length; i++) {
    // Pop while current is greater than stack top
    while (stack.length && nums[i] > nums[stack[stack.length - 1]]) {
      result[stack.pop()] = nums[i]; // found the answer!
    }
    stack.push(i);
  }

  return result; // remaining in stack → -1 (no greater)
}

Code: Daily Temperatures

// How many days until a warmer temperature?
function dailyTemperatures(temps) {
  const result = new Array(temps.length).fill(0);
  const stack = [];

  for (let i = 0; i < temps.length; i++) {
    while (stack.length && temps[i] > temps[stack[stack.length - 1]]) {
      const idx = stack.pop();
      result[idx] = i - idx; // days between
    }
    stack.push(i);
  }
  return result;
}

Keywords: "next greater", "next smaller", "previous greater", "daily temperatures", "stock span", "largest rectangle", "trapping rain water".

Pattern: Min Stack

Design a Stack with O(1) getMin()

Support push, pop, top, AND getMin() — all in O(1) time.

class MinStack {
  constructor() {
    this.stack = [];    // stores [value, currentMin]
  }

  push(val) {
    const min = this.stack.length === 0
      ? val
      : Math.min(val, this.getMin());
    this.stack.push([val, min]); // store PAIR
  }

  pop()    { this.stack.pop(); }
  top()    { return this.stack[this.stack.length - 1][0]; }
  getMin() { return this.stack[this.stack.length - 1][1]; }
}
The trick: Each entry stores [value, minSoFar]. When you pop, the min info pops with it. The current min is always at the top.

Pattern: Evaluate Expression

Reverse Polish Notation (Postfix)

Evaluate ["2","1","+","3","*"] → (2+1)*3 = 9

function evalRPN(tokens) {
  const stack = [];

  for (const token of tokens) {
    if ('+-*/'.includes(token)) {
      const b = stack.pop(), a = stack.pop();
      if (token === '+') stack.push(a + b);
      if (token === '-') stack.push(a - b);
      if (token === '*') stack.push(a * b);
      if (token === '/') stack.push(Math.trunc(a / b));
    } else {
      stack.push(Number(token)); // number → push
    }
  }

  return stack[0];
}
Rule: Numbers → push. Operators → pop two, compute, push result. Simple!

Basic Calculator (Infix with Parentheses)

function calculate(s) {
  const stack = [];
  let num = 0, sign = 1, result = 0;

  for (const ch of s) {
    if (ch >= '0' && ch <= '9') {
      num = num * 10 + Number(ch);
    } else if (ch === '+' || ch === '-') {
      result += sign * num;
      num = 0;
      sign = ch === '+' ? 1 : -1;
    } else if (ch === '(') {
      stack.push(result); stack.push(sign);
      result = 0; sign = 1;
    } else if (ch === ')') {
      result += sign * num; num = 0;
      result *= stack.pop(); // sign before paren
      result += stack.pop(); // result before paren
    }
  }
  return result + sign * num;
}

The Call Stack & DFS

Every Function Call Is a Stack Push

When you call a function, it's pushed onto the call stack. When it returns, it's popped. That's why recursion works — and why too much recursion causes Stack Overflow!

// The call stack IS a stack
function factorial(n) {
  if (n <= 1) return 1;   // base case → start popping
  return n * factorial(n - 1); // push new call
}
// factorial(4) pushes: [4] → [4,3] → [4,3,2] → [4,3,2,1]
// then pops:           [4,3,2] → [4,3] → [4] → result

DFS with Explicit Stack

// Convert recursive DFS → iterative with stack
function dfsIterative(root) {
  if (!root) return [];
  const stack = [root];
  const result = [];

  while (stack.length) {
    const node = stack.pop();
    result.push(node.val);
    if (node.right) stack.push(node.right); // right first
    if (node.left)  stack.push(node.left);  // so left pops first
  }
  return result;
}
DFS = Stack. BFS = Queue. This is the fundamental rule. Recursive DFS uses the call stack implicitly. Iterative DFS uses an explicit stack. BFS uses a queue.

Pattern: Decode String

The Problem

3[a2[bc]] → abcbcabcbcabcbc. Nested encoding — a perfect stack problem.

function decodeString(s) {
  const countStack = [];
  const strStack = [];
  let current = '';
  let k = 0;

  for (const ch of s) {
    if (ch >= '0' && ch <= '9') {
      k = k * 10 + Number(ch);
    } else if (ch === '[') {
      countStack.push(k);
      strStack.push(current);
      current = ''; k = 0;
    } else if (ch === ']') {
      const repeat = countStack.pop();
      current = strStack.pop() + current.repeat(repeat);
    } else {
      current += ch;
    }
  }

  return current;
}
Two stacks: One for counts, one for the string built so far. When you hit [, save context. When you hit ], restore context and repeat.

Tricks & Techniques

Trick: Two Stacks → Queue

class QueueFromStacks {
  constructor() { this.inStack = []; this.outStack = []; }

  enqueue(val) { this.inStack.push(val); }

  dequeue() {
    if (!this.outStack.length)
      while (this.inStack.length)
        this.outStack.push(this.inStack.pop()); // reverse!
    return this.outStack.pop();
  }
}
Amortized O(1): Each element gets pushed and popped twice total (once per stack). Amortized = O(1) per operation.

Trick: Remove Adjacent Duplicates

function removeDuplicates(s) {
  const stack = [];
  for (const ch of s) {
    if (stack.length && stack[stack.length - 1] === ch)
      stack.pop();   // matching adjacent → remove
    else
      stack.push(ch);
  }
  return stack.join('');
}

Trick: Asteroid Collision

function asteroidCollision(asteroids) {
  const stack = [];
  for (const a of asteroids) {
    let alive = true;
    while (alive && a < 0 && stack.length && stack[stack.length-1] > 0) {
      const top = stack[stack.length - 1];
      if (top < -a) { stack.pop(); }       // top smaller, destroyed
      else if (top === -a) { stack.pop(); alive = false; } // both gone
      else { alive = false; }               // current destroyed
    }
    if (alive) stack.push(a);
  }
  return stack;
}
